Atomic fetch and add operation for pointers on a global address.

Parameters

valueThe value to add to the object
orderMemory synchronization ordering for this operation
Template Parameters
TThe type of the pointer to operate upon

Returns
The value of the pointer BEFORE the add operation

This function will perform an atomic (*obj)+=(*value) operation. This is an overload for pointer types, as pointer arithmetic works differently than normal integer and floating point arithmetic.

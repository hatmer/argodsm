Detailed Description

template<typename T, typename MemoryPool, typename LockType>
class argo::allocators::generic_allocator< T, MemoryPool, LockType >

generic template for memory allocators

Template Parameters
Ttype to allocate
MemoryPoolthe type of memory pool to allocate from
LockTypelock type used to protect the memory pool

This template can be used for many different kinds of memory allocators in ArgoDSM: The underlying memory pool is currently either a preallocation pool or a large piece of distributed shared memory. Locking is only required when the memory pool otherwise would experience data races.

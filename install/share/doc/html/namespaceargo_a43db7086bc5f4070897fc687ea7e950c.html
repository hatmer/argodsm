The function might choose to initialize or synchronize, depending on the type constructed or the arguments provided by the user.

  1. Initialization is performed by default.
  2. If the type is trivial, then no initialization is performed.
  3. If the user has provided constructor arguments, then initialization is performed.
  4. The user can explicitly override these defaults using the appropriate allocation parameters (allocation).
  5. Synchronization is performed after initialization.
  6. The user can override the behaviour by using the allocation parameters.

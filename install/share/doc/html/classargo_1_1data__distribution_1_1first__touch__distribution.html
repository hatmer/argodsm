Detailed Description

template<int instance>
class argo::data_distribution::first_touch_distribution< instance >

the first-touch data distribution

gives ownership of a page to the node that first touched it.

Note
if a node's backing store size is not sufficient, it passes the ownership of the page to a node that can host it.

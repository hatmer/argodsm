Detailed Description

template<int instance>
class argo::data_distribution::base_distribution< instance >

the base data distribution class

this is the parent class from which all the memory policies inherit in order to implement their own homenode and local_offset functions.

Template Parameters
instanceused to statically allow for multiple instances
Note
all functions are defined on char* only, as this guarantees a fixed memory base unit of size 1.

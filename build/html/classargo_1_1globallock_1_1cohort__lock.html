Detailed Description

a global 'cohort' lock - needs to be called collectively

Locks in levels and tries to hand over the lock as locally as possible.

Warning
Do not allocate this lock on the global memory. It contains data that need to be node local. Also, keep in mind that the constructor calls conew_, so the constructor should be called by all the nodes at the same time.

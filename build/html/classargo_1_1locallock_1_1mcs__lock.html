Detailed Description

MCS mutual exclusion lock (intranode)

This lock implementation is based on a C implementation by Kjell Winbland https://github.com/kjellwinblad/qd_lock_lib This lock only works for individual nodes and not across multiple nodes.

Warning
The MCS need to be locked and unlocked by the same thread.

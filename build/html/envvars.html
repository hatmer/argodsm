ArgoDSM - Software Distributed Shared Memory
Environment Variables

ARGO_MEMORY_SIZE
request a specific memory size in bytes

This environment variable is only used if argo::init() is called with no argo_size parameter (or it has value 0). It can be accessed through argo::env::memory_size() after argo::env::init() has been called.

ARGO_CACHE_SIZE
request a specific cache size in bytes

This environment variable is only used if argo::init() is called with no cache_size parameter (or it has value 0). It can be accessed through argo::env::cache_size() after argo::env::init() has been called.

ARGO_WRITE_BUFFER_SIZE
request a specific write buffer size in cache blocks

This environment variable defaults to 512 cache blocks if not specified. It can be accessed through argo::env::write_buffer_size() after argo::env::init() has been called.

ARGO_WRITE_BUFFER_WRITE_BACK_SIZE
request a specific write buffer write back size in cache blocks.

This environment variable defaults to 32 cache blocks if not specified. It can be accessed through argo::env::write_buffer_write_back_size() after argo::env::init() has been called.

ARGO_ALLOCATION_POLICY
request a specific allocation policy with a number

This environment variable can be accessed through argo::env::allocation_policy() after argo::env::init() has been called.

ARGO_ALLOCATION_BLOCK_SIZE
request a specific allocation block size in number of pages

This environment variable can be accessed through argo::env::allocation_block_size() after argo::env::init() has been called.

ARGO_LOAD_SIZE
request a specific load size in number of pages

This environment variable determines the maximum amount of DSM pages that are fetched on each remote load operation. It can be accessed through argo::env::load_size() after argo::env::init() has been called.

Detailed Description

This file provides the dynamic allocator for ArgoDSM.

Copyright
Eta Scale AB. Licensed under the Eta Scale Open Source License. See the LICENSE file for details.

There are two types of dynamic allocators provided by this file. Firstly, there is the global allocator, which provides memory directly from the distributed shared memory pool. However, this allocation is very slow, and thus is only used by preallocation memory pools. Secondly, the dynamic allocator maintains a preallocation pool on each ArgoDSM node, and can therefore allocate memory much faster. As preallocation pools are not threadsafe, the dynamic allocator must be protected within the ArgoDSM node's scope.
